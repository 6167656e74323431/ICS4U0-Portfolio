ICS4U0 Final Project

Back box cover for the game we designed.

For the final project for this course we had to work in partners to make a game of some kind. This game could be anything as long as it had the theme, overcoming adversities. The adversity that we chose to do was time-management.

The game we came up with was this desktop-simulator game where you had to manage your time while doing many different tasks. We figured this would be a good idea as teenagers these days are perfectly willing to do the work (I know from experience), but they just don't know how to manage their time.

The final project was done in partners, of which I chose Larry, who's name is on many other files on this portfolio (I have worked with him before). I think this was a great choice, as he is a reliable partner, who always does his work.

The entire game is, or should be on the github that we created for an imaginary company which can be found here. We are so dedicated to this project, that we even reserved a (free) domain, just so our package structure can follow proper Java conventions. The domain (http://educationallydesigned.ml) just links back to the github organization.
